# 📊 MIS Dashboard & SLA Forecasting (Excel Project)

## 🚀 Overview

This project demonstrates an end-to-end MIS (Management Information System) dashboard along with SLA forecasting using Microsoft Excel. It focuses on tracking business performance metrics and predicting future workload trends.

## 🎯 Objectives

* Build an interactive MIS dashboard
* Track KPIs like SLA, CSAT, Productivity, Attendance
* Perform workload forecasting using historical data
* Support data-driven decision making

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables & Charts
* Slicers
* FORECAST.LINEAR
* Advanced Excel Functions

## 📊 MIS Dashboard Features

* KPI Cards (SLA, CSAT, Productivity)
* Department-wise performance analysis
* Employee productivity tracking (Top 10)
* Attendance distribution
* Interactive filters (Department, Shift)

## 📈 Forecasting Analysis

* SLA and workload trend analysis
* Forecasted future ticket volume using Excel
* Visual comparison of actual vs predicted values
